## 搭建开发环境

下载第三方库

```shell
# 异步框架
pip install aiohttp
# 前端模板引擎
pip install jinja
# MySQL异步驱动程序
pip install aiomysql
```
目录结构

```
|__backup 备份
|__conf   配置文件
|__dist   打包目录
|__www    Web目录
    |__static  静态资源
    |__templates  模版文件
```
